HTML Extra Markup Reference Cheatsheet

HTML Tags

Tag Name Description
<div> Division A tag that defines a division or a section in an HTML document. Also used as a container for HTML elements, which can then be styled with CSS or manipulated with JavaScript. It is easily styled by using the class or id attribute. Any sort of content can be put inside!
<span> Span This tag is an inline container used to mark up a part of a text, or a part of a document. It can easily be styled by CSS or manipulated with JavaScript using the class or id attribute. Much like the <div>element, but <div> is a block-level element and <span> is an inline element.
<iframe> iframe This tag specifies an inline frame. It is used to embed another document within the current HTML document. Tip: Use CSS to style the <iframe>. It is a good practice to always include a title attribute
<meta> Metadata This tag defines metadata about an HTML document. Metadata is data (information) about data. These tags always go inside the <head> element, and are typically used to specify character set, page description, keywords, author of the document, and viewport settings.

HTML Attributes

Attribute Name Description
id Identify It specifies a unique id for an HTML element (the value must be unique within the HTML document). It is a Global Attribute, and can be used on any HTML element. It is most used to point to a style in a style sheet, and by JavaScript (via the HTML DOM) to manipulate the element with the specific id.
class Classnames It specifies one or more classnames for an element and mostly used to point to a class in a style sheet. However, it can also be used by a JavaScript (via the HTML DOM) to make changes to HTML elements with a specified class.
src Source The src attribute specifies the URL of the source.
height Height The height attribute specifies the height in pixels.
width Width The width attribute specifies the width in pixels.

HTML Elements

Element Name Description
Block Block Level Block level elements will always appear to start on a new line in the browser window. Examples are <h1>, <p>, and <li>.
Inline Inline Inline Elements will always appear to continue on the same line as their neighbouring elements. Examples are <a>, <b>, <em>, and <img>.
